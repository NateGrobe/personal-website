Projects

  • Maze Generator

    Uses a depth first algorithm to randomly generate a maze. The size of the maze is decided by the user.

    The whole project was written in python. To visualize the algorithm as it runs, the Turtle module was used.

    The blue demonstrates the backtracking done by the algorithm when a dead end is reached.


  • Pixel Art Generator

    Takes an image and generates a pixelated version of the image using Python and the Pillow library. The image is pixelated by accessing 10x10 squares of pixels from the original image, averaging their RGB values, and replacing the original pixel values with the new RGB values.


  • Rubik's Cube

    Automatically or manually scrambles a rubik's cube and then displays each face of the cube in the terminal after scrambling.


  • Dashboard

    Tracks assignments for school courses. Sorts them from a csv file based on course, completion status and due date. The web-app was written in Python using the Flask library.
